Общие требования 
Приложение разрабатывается на WPF и должно содержать:
• главное меню
• контекстное меню
• обработку клавиатуры
• использование MessageBox (не менее 3 разных типов)
• обработку события закрытия окна
Дополнительные требования:
• логика не должна дублироваться (одни и те же действия вызываются из разных мест)
• код должен быть структурирован (вынос логики в отдельные методы)
• корректная обработка пользовательских действий
Временные изменения
Реализовать режим предпросмотра:
Требования:
• изменение применяется временно
• пользователь подтверждает или отменяет
• при отмене — возврат к предыдущему состоянию
